# github-client-branch-skill

A reusable AgentSkill for creating and pushing dedicated client customization branches from a company name and a GitHub repository URL.

## Contents

- `skill/github-client-branch/` - source skill files
- `dist/github-client-branch.skill` - packaged skill artifact

## What it does

Input:
- company name
- GitHub repository URL

Output:
- generated branch name
- GitHub branch URL

Example branch style:
- `client/miaoxing-youpin`
